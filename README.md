# Thoughtless

### One Liner:

A "Kingdom" like side scrolling game where you save cute animals from mindrot.

### Further poetic description:

All cute animals are consumed by thoughtless mindrot. Parents exiled you because you are useless. 3 rats start following you. Together you defeat a bunny. Now 3 rats and a bunny follow you. Your journey begins. Free animals and the world from mindrot!

### More detailed description:

This is a game really similar to "Kingdom" where you organise a troop build a base and face off against waves of enemies at night and then turn defeated enemies into allies in the day :] not too complicated for now ;-; but its something :3
